* AI Logo Generator

An AI-powered Logo Generator that creates multiple SVG logo concepts based on brand details using Google's Gemini API.

---

* Features

- Generate 3 unique logo concepts
- AI-generated SVG logos
- Logo rationale and symbol meaning
- Color recommendations
- Download SVG logos
- Responsive user interface

---

* Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Google Gemini API
- dotenv
- CORS

* How to Use

1. Enter Brand Name.
2. Enter Brand Description.
3. Enter Industry Type.
4. Enter Target Audience.
5. Click Generate Logos.
6. View 3 generated logo concepts.
7. Download any SVG logo.
